# PyProg2026Lab4

Лабораторная работа 4 (P3123): классификация дефолта по кредиту — Scikit-Learn.

## Файлы

- `s2p1_predict_credit_default_tasks.ipynb` — борд с заполненными ячейками и самостоятельным заданием
- `training_data.csv`, `test_data.csv` — данные Kaggle *Give Me Some Credit*

## Запуск

```powershell
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
jupyter notebook s2p1_predict_credit_default_tasks.ipynb
```

Открыть ноутбук и выполнить **Run All** (нужны CSV в этой же папке).
